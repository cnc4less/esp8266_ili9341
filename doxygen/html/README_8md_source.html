9 @par Description
10  * ESP8266 support libraries
11  * All of the code in this project, except where noted is written by me
12  * Analog Devices ADF4351 Wideband Synthesizer driver and test code
13  * Control is via serial terminal that works while the other demos run
14  * adf4351 123456000 10000
15  * frequency, and channel spacing used in calculations
16  * adf4351_scan 144e6 145e6 10000
17  * start frequency, end frequency, channel spacing
18  * ILI9341 display driver
19  * Mutiple custom size window support
20  * Window scrolling
21  * readPixel() function suported
22  * The specific display this was tested on is a TM022HDH26 display I got from ebay.
23  * Not limited by ILI9341 hardware scroll restrictions.
24  * BDF compiler and fonts - creates C code used in the display drivers
25  * Wireframe Viewer
26  * Code generator for Earth coastline data
27  * I created two wireframe demos
28  * Cube wireframe dataset
29  * Earth coastline dataset - wireframe view still needs hidden line removal option
30  * CORDIC C code generator and 3D transformation code support functions use by wireframe viewer code
31  * Small PRINTF with full floating point support - much smaller then GNU full verson along
32  * ATOF and various integer conversions
33  * WEB server using SD CARD with CGI processing - files and CGI results can be ANY SIZE!
34  * Example web site for testing
35  * WEB server can update TFT display
36  * Simple door sign status update using web page - see html/msg.cgi and web/web.c for code
37  * Network server client example for display updates
38  * Uart network server client for serial uart to network bridge.
39  * Generic queue handling code
40  * Uart driver.
41  * POSIX wrappers for FatFS - provides unix file I/O operations
42  * Includes FatFS SD card support by ChaN
43  * POSIX time functions and RTC set with NTP
44  * Multiple timers used by rtc and time functions
45  * HSPI code that can handle multiple devices each with varying clock frequencies
