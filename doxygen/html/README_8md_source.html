9 @par Description
10  * ESP8266 support libraries
11  * All of the code in this project, except where noted is written by me
12  * Analog Devices ADF4351 Wideband Synthesizer driver and test code
13  * Control is via serial terminal that works while the other demos run
14  * adf4351 123456000 10000
15  * Frequency, and channel spacing used in calculations
16  * adf4351_scan 144e6 145e6 10000
17  * start frequency, end frequency, channel spacing
18  * ILI9341 display driver
19  * Multiple custom size windows
20  * Custom fonts
21  * Window scrolling
22  * readPixel() function supported
23  * The specific display this was tested on is a TM022HDH26 display I got from eBay.
24  * Not limited by ILI9341 hardware scroll restrictions.
25  * xpt2046 touch controller
26  * filtered read of XY
27  * debounced key press read of queued events
28  * 5 point calibration routines
29  * BDF FONT compiler tools and fonts sets
30  * Tools to creates C code used in the display drivers
31  * Wireframe Viewer
32  * Code generator for Earth coastline data
33  * I created two wireframe demos
34  * Cube wireframe dataset
35  * Earth coastline dataset - wireframe view still needs hidden line removal option
36  * CORDIC C code generator and 3D transformation code support functions use by wireframe viewer code
37  * Small PRINTF with full floating point support - much smaller then GNU full version along
38  * Additional number IO functions, ATOF etc
39  * WEB server using SD CARD with CGI processing - files and CGI results can be ANY SIZE!
40  * Example web site for testing
41  * WEB server can update TFT display
42  * Simple door sign status update using web page - see html/msg.cgi and web/web.c for code
43  * Network server client example for display updates
44  * Uart network server client for serial uart to Network Bridge.
45  * Generic queue handling code
46  * ESP8266 support for FatFS by ChaN 2016

419  * MOSI/GPIO13
420  * CLK/GPIO14
421  * CHIP_EN ---1K--- VCC3.3V
422  * CH_PD ---1K--- VCC3.3V
423  * REST ---1K--- VCC3.3V Ground REST to reset ESP8266 and ILI9341
424  * GPIO 2 ---1K--- VCC3.3V
425  * GPIO 0 ---1K--- VCC3.3V Ground GPIO 0 to to enable flashing
426  * GPIO 15 ---1K--- GND Need pull down for boot
427  * GPIO 14 ---1K--- GND Keeps floating pins from generating false clocks on attached devices
428  * GND ------------ GND
429  * T_OUT ------ + T_OUT is connected to varaible VR100K
430  |
431  VCC--330K--VR100K-- GND Variable VR100K is connected to 330K then VCC
432  Max voltage to T_OUT pin is VCC * 100K / (330K+100K)
433 
434  * FTDI232 USB/TTL ESP8266
435  * TXD ----1K-------- RXD
436  * RXD -------------- TXD
437  * RTS ---|<--------- REST Optional - Diode with Cathode to RTS, Anode to ESP8266 Reset
438  * DTR ---|<----1K--- GPIO 00 Optional - Diode with cathode to DTR, Anode to 1K to ESP8266 GPIO00 for flashing
439  * GND -------------- GND
440  * Note: if you use the optional DTR,RTS conneections you have to use miniterm script provided
